Downloading SPA JS

Direct Download

Since SPA JS is built on Handlebars libraries, download the dependency libraries from the respective sites into respective xlib folder.

(To locally download these files, right-click on the link and select "Save as..." from the menu)

For jQuery (optional for SPA.js v2.87.0+) Download Latest or visit https://jquery.com/download/

For Handlebars Download Latest or visit https://handlebarsjs.com/installation/#downloading-handlebars

For SPA JS Download Latest (Production) or Download Latest (Development)
