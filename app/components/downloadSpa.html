Downloading SPA JS

Using Bower

If “bower” is installed, use “bower” CLI to download SPA JS along with its dependencies.

Results in

      

        bower_components
        Jquery
        Handlebars
        spa

Direct Download

Since SPA JS is built on jQuery and Handlebars libraries, download the dependency libraries from the respective sites.

For jQuery Download Latest or visit http://jquery.com/download/

For Handlebars Download Latest or visit http://handlebarsjs.com/installation.html

For SPA JS Download Latest (Production) or Download Latest (Development) or visit https://spa.js.org or http://spajs.org 

      

        Xlib
        Jquery
        jquery.min.js
        Handlebars
        handlebars.min.js
        spa
        spa-bundle.min.js

Using CDN

If the application is hosted on the server, which has internet connection, then it’s recommeded to use CDN

In addition to dependent libraries, any 3rd party libraries like Bootstrap, Foundation, Semantic UI, Moments, DataTables, etc. can be dowloaded and included in the project using bower or npm or yarn or direct download.
