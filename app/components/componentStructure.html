SPA Component

SPA Component is a partial view in a Single Page Application. The partial view is

  • - Plain HTML Template
  • - May bind data using Handlebars syntax
  • - May bind events (specified in its javascript file) to handle user interactions like clicks.

Component Structure

A SPA component is sub-folder with .html and .js files in app/components folder.

        
          app
          components
          componentX
          componentX.html
          componentX.js
          header
          header.html
          header.js
          footer
          footer.html
          footer.js
          main
          main.html
          main.js
          css
          js
          language
          xlib
          index.html
  • NOTE:
  • 1. Name of FOLDER, HTML file and JS file should be same (case sensitive)

  • 2. Component Name:
    a. Case Sensitive
    b. BEGINS with Alphabet
    c. ONLY Alpha-numberic characters are allowed
    d. NO special characters are allowed

Using CLI

Use spa-cli to create App and Component Structure quickly.
