Routing

Though client-side Routing is an important feature of SPA, it is not a must feature. SPA can be built without routing. However, if your project requires client-side Routing, consider the following libraries or any other alternate according to your needs.

  • Page.js

    Tiny Express-inspired dependency free client-side routing library. This Quick Guide will help you to get started.

  • director

    A dependency free routing library works in browser. Simplifies the development of Single Page Apps. Visit its GitHub repo for more details.
