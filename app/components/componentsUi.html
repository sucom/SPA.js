Event Binding in Template

Using HTML native event attributes, we can bind a function defined in component’s script ( componentX.js ) file to any HTML element in the template. 

							
								
    
									
  1. 
										<
										button
										 onclick=
										"{$}submitForm();"
										>Submit</button>
