The if block helper

Use the if helper to conditionally render a block. If its argument returns false, undefined, null, "", 0 or [ ], Handlebars will not render the block

When used with an empty ( { } ) context, author will be undefined, resulting in:

When using a block expression, (optional) specify a template section to run if the expression returns a falsy value. The section, marked by {{else}} is called an "else section".
